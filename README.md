## 🚀 Installation

<div align="center">

### Quick Start Guide

</div>

```bash
# 📥 Clone the repository
git clone https://github.com/yourusername/procalc-advanced-calculator.git
cd procalc-advanced-calculator

# 📦 Install dependencies
npm install

# 🚀 Start the development server
npm start

# 📱 Scan QR code with Expo Go app
```

### Alternative Methods

</div>
